# FumoAgent

[中文](#中文) | [English](#english) | [日本語](#日本語)

---

## 中文

FumoAgent 是一个基于 Gio UI 与 Tauri 框架开发的桌面宠物/看板娘程序。项目旨在打造一个界面干净、资源占用极低的轻量化桌面陪伴工具，并提供丰富的本地化语音自定义功能。

### 项目特点
- **轻量低耗**：采用 Gio UI + Tauri 构建，保持极低的本地系统资源占用。
- **本地化语音**：语音合成完全由本地引擎驱动，支持丰富的本地化语音自定义，摆脱对云端语音服务的依赖。
- **配置简单**：在运行过程中，仅需配置可用的 OpenAI 协议大语言模型 API 即可实现智能对话。

### 核心功能
- **桌面陪伴与简单交互**：日常驻留桌面，支持基础的鼠标点击与交互响应。
- **智能对话**：支持对接任意兼容 OpenAI 标准接口的 AI 语言模型。
- **多语音引擎支持**：
  - **VoiceVox**：[项目主页](https://voicevox.hiroshiba.jp/)
  - **AquesTalk**：[项目主页](https://www.a-quest.com/products/aquestalk.html)
  - **Open JTalk**：[项目主页](https://open-jtalk.sourceforge.net/) *(注：目前在部分系统环境下可能存在一些兼容性问题，正在优化中)*

## English

FumoAgent is a desktop pet / kanban girl application developed using the Gio UI and Tauri frameworks. The project aims to create a lightweight desktop companion with a clean interface, minimal resource consumption, and rich localized voice customization features.

### Features
- **Lightweight & Low Consumption**: Built with Gio UI + Tauri to maintain extremely low local system resource usage.
- **Localized Voice**: Voice synthesis is fully driven by local engines, supporting rich localized voice customization and eliminating dependency on cloud voice services.
- **Simple Configuration**: During operation, it only requires configuring a compatible OpenAI protocol LLM API to enable intelligent dialogue.

### Core Functions
- **Desktop Companion & Interaction**: Stays on the desktop, supporting basic mouse clicks and interaction responses.
- **Intelligent Dialogue**: Supports connecting to any AI language model compatible with the standard OpenAI API.
- **Multiple Voice Engine Support**:
  - **VoiceVox**: [Project Homepage](https://voicevox.hiroshiba.jp/)
  - **AquesTalk**: [Project Homepage](https://www.a-quest.com/products/aquestalk.html)
  - **Open JTalk**: [Project Homepage](https://open-jtalk.sourceforge.net/) *(Note: Currently there may be some compatibility issues in certain system environments, which are being optimized.)*

## 日本語

FumoAgent は、Gio UI と Tauri フレームワークを用いて開発されたデスクトップペット・看板娘アプリケーションです。クリーンなインターフェースと極めて低いリソース消費を実現した軽量なデスクトップ相棒ツールを目指しており、豊富なローカル音声カスタマイズ機能を提供します。

### 特徴
- **軽量・低リソース**：Gio UI + Tauri を採用し、ローカルシステムのシステムリソース消費を極限まで抑えています。
- **ローカル音声**：音声合成は完全にローカルエンジンによって駆動され、豊富なローカル音声カスタマイズに対応。クラウドの音声サービスに依存しません。
- **簡単な設定**：実行時は、OpenAI プロトコルと互換性のある AI 言語モデルの API を設定するだけで、インテリジェントな会話が可能になります。

### 主な機能
- **デスクトップの相棒と簡易インタラクション**：日常的にデスクトップに常駐し、基本的なマウスクリックやインタラクションの応答に対応します。
- **インテリジェントな会話**：OpenAI 標準インターフェースと互換性のある任意の AI 言語モデルとの連携をサポート。
- **マルチ音声エンジン対応**：
  - **VoiceVox**：[プロジェクトページ](https://voicevox.hiroshiba.jp/)
  - **AquesTalk**：[プロジェクトページ](https://www.a-quest.com/products/aquestalk.html)
  - **Open JTalk**：[プロジェクトページ](https://open-jtalk.sourceforge.net/) *(注：現在、一部のシステム環境において互換性の問題が発生する場合があり、最適化を進めています。)*
